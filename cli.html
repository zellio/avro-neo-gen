CLI

avro-neo-gen

Avro NeoGen: Typed code generation for Avro schemas and protocols.

avro-neo-gen [OPTIONS] COMMAND [ARGS]...

Options

-l, --log-level <log_level>

Log level

Default

info

Options

critical | error | warning | info | debug

-L, --log-format <log_format>

Log format

Default

plain

Options

plain | json

Environment variables

AVRO_NEOGEN_LOG_LEVEL

Provide a default for -l

AVRO_NEOGEN_LOG_FORMAT

Provide a default for -L

compile

Compile Avro schema files into a typed Python module.

avro-neo-gen compile [OPTIONS]

Options

-a, --avro-source-directory <avro_source_directory>

Required Directory which contains Avro schema and protocol definitions.

-p, --python-target-directory <python_target_directory>

Required Base directory to write out generated file tree

Default

./build

-D, --avro-driver <avro_driver>

Required Driver for managing avro data in generated code.

Default

apache_avro_binary_driver

-d, --dry-run

Generate code without writing files to disk.

Default

False

-f, --force

Force generation of Python library.

Default

False

Environment variables

AVRO_NEOGEN_COMPILE_SOURCE_DIRECTORY

Provide a default for -a

AVRO_NEOGEN_COMPILE_TARGET_DIRECTORY

Provide a default for -p

AVRO_NEOGEN_COMPILE_DRIVER

Provide a default for -D

AVRO_NEOGEN_COMPILE_DRY_RUN

Provide a default for -d

AVRO_NEOGEN_COMPILE_FORCE

Provide a default for -f

list-drivers

List provided Avro protocol drivers.

avro-neo-gen list-drivers [OPTIONS]
