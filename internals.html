On load, the corelib updates Python‚Äôs module path to include the root of the generated source tree. This is done so the avro_neo_gen.core namespace can be properly loaded.

import pathlib

    sys.path.append(str(lib_path))

class avro_neo_gen.avro_schema.AvroSchema(schema: T)[source]ÔÉÅ

Bases: Generic[T]

Wrapper class for Apache Avro Schema types.

Provide additional functionality for avro.schema.Schema.

AvroSchema extends the Apache Avro interface to smooth out differences in the underlying types so the are easier to operate on.

Importantly AvroSchema guarantees that name and namespace are safe to call, returning None when the field is missing.

Parameters

schema (Union[AvroSchema, T]) ‚Äì An avro.schema.Schema instance or an AvroSchema instance to be wrapped.

Raises

TypeError ‚Äì If provided schema

Note

Class can be safely called on AvroSchema, which will return a new instance which re-wraps the previously wrapped value.
